Verwendung von Open Source Daten

In diesem Tutorial lernen Sie, wie Sie Open Source Daten effektiv nutzen können. Wir konzentrieren uns besonders auf Satellitendaten der ESA BIOMASS-Mission und zeigen Ihnen Schritt für Schritt, wie Sie:

  • Zugang zu den Daten erhalten
  • Die Daten herunterladen und verarbeiten
  • Grundlegende Analysen durchführen
  • Ihre eigenen Referenzdaten sammeln und beitragen

How to Use Open Source Data

In this tutorial, you'll learn how to effectively use open source data. We focus particularly on satellite data from the ESA BIOMASS mission and show you step by step how to:

  • Access the data
  • Download and process the data
  • Perform basic analyses
  • Collect and contribute your own reference data
